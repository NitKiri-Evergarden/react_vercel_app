# Harry Potter Books App

Aplicação React que consome a API pública de livros de Harry Potter.

## Tecnologias
- React 18
- Vite 7
- Node.js 20.19
- Lucide React (ícones)
- Tailwind CSS

## Instalação
```bash
npm install
npm run dev
```
